Chapter 6 - UFO Game

This chapter will cover the final project for the course. The project will be a UFO game where the user will try to catch as many UFOs as possible. You will be provided with a sample code that doesn't complete the requirements of the game. You will need to complete the code to make the game work.

We will need to import View, Text, TouchableOpacity, and useEffect and useState. We can import these components from the react-native and react package.

import React, {useState, useEffect} from 'react';
import {View, Text, TouchableOpacity} from 'react-native';

Random​

The first thing we need to do is to generate a random location for the UFO.

Randomness is an important concept in programming and can be used in many ways. It can be used to generate numbers (e.g., roll a dice), shuffle data, etc. We will use the Math.random() function to generate a random number between 0 and 1. If we multiply this number by 300, we will get a number between 0 and 300. If we add 1 to this number, we will get a number between 1 and 301. Then, if we use the Math.floor() function to round the number down to the nearest integer, we will get a integer between 1 and 300.

For our UFO game, we will generate a random location for the UFO by generating a random number between 1 and 300 for the x position and a random number between 1 and 500 for the y position.

Here is a sample code that generates a random location for the UFO:

const randomLocation = () => {
  const x = Math.floor(Math.random() * 300) + 1;
  const y = Math.floor(Math.random() * 500) + 1;
  return {x, y};
};

State​

We will need to use the useState hook to keep track of the score and the UFO's position. The score will be an integer, and the UFO's position will be an object with x and y properties.

const [score, setScore] = useState(0);
const [ufoPosition, setUfoPosition] = useState(randomLocation());

Handle UFO Click​

We will need to create a function that will be called when the user clicks on the UFO. This function will update the score and generate a new random location for the UFO.

const handleUfoPress = () => {
  setScore(score + 1);
  setUfoPosition(randomLocation());
};

Game Over Effect​

We will need to use the useEffect hook to check if the score has reached 50. If it has, we will display an alert to the user.

useEffect(() => {
  if (score >= 50) {
    alert('Game Over!');
  }
}, [score]);

View and styles​

We will need to create a View component that will contain the score and the UFO. We will also need to create a Text component that will display the score. We will use the TouchableOpacity component to display the UFO.

We will set the position of the UFO to absolute and set the left and top properties to the randomly generated x and y values. We will also need to add an onPress event handler to the TouchableOpacity component that will call the handleUfoPress function, which will update the score and generate a new random location for the UFO.

<View style={{flex: 1}}>
  <Text
    style={{
      textAlign: 'center',
      fontSize: 24,
      marginVertical: 10,
    }}>
    Score: {score}
  </Text>
  <TouchableOpacity
    style={{
      position: 'absolute',
      left: ufoPosition.x,
      top: ufoPosition.y,
    }}
    onPress={handleUfoPress}>
    <Text style={{fontSize: 30}}>🛸</Text>
  </TouchableOpacity>
</View>

UFO Game with Timer​

In this section, we will create a UFO game that will generate a new UFO every second. If the user clicks on the UFO, the score will increase by 1. If the user does not click on the UFO, the UFO will disappear and the next UFO will appear after 1 second. The game will end when the score reaches 50.

In this capstone project, you are going to use what you have learned in this course to create the game. You will need to use the useState hook to keep track of the score and the UFO's position. You will also need to use the useEffect hook to generate a new UFO every second. You will need to use the setInterval function to generate a new UFO every second.
