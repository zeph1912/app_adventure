Chapter 3 - Scroll

This chapter will introduce you to more components that you can use to build your app, and how to style them. Specifically, you will learn how to use the ScrollView component to display lots of content in a limited space, and how to use the FlatList component to display a list of data. You will also learn how to style your app using the StyleSheet component.

Scroll​

ScrollView is a useful feature in app development that helps us to display lots of content in a limited space. The ScrollView is a container that allows us to scroll through multiple components and views, making it possible for us to display a variety of items on a single screen.

In other words, think of the ScrollView as a large container that holds all of the content you want to display. The content can be different types of components or views, such as images, text, or buttons. When the content exceeds the available space, the ScrollView allows us to scroll through it.

One of the great things about ScrollView is that it allows us to have a mix of different types of content within the same container. For example, we can have images, text, and buttons all within the same ScrollView. This is useful when we have a lot of content to display and want to keep it all in one place.

List​

In React, both lists and scrolls can help us display lots of information on a screen. However, they work a little differently.

A list is like a paper list you might make. It shows all the items one after the other, and you have to scroll down to see the ones at the bottom. Think of a grocery list where you write each item one below the other.

A scroll, on the other hand, is like a window into a bigger space. You can see part of the space through the window, but you have to move the window to see other parts. Imagine looking through a small window into a huge room. You can only see a part of the room at a time, and you need to move the window around to see different parts.

Styling with Flex​

Have you ever tried fitting many things into a small space, like trying to fit all your toys into a small box? That can be hard, right? Well, imagine you have a big box and you want to fit some things in it, but you're not sure how much space each thing will need. That's where flex comes in!

When we use flex in a component's style, we can make it expand or shrink based on how much space is available. Think of it like a rubber band - the more you stretch it, the longer it gets. In the same way, when we give a component a higher flex value, it will take up more space compared to other components with the same View container.

For example, if you have three boxes next to each other and you want them to share the space evenly, you can give each box a flex value of 1. This means each box will take up the same amount of space. But if you want one box to be bigger than the others, you can give it a higher flex value. The higher the "flex" value, the more space the box will take up compared to the other boxes.

In the example below, we have a View container with three boxes inside of it. The first box has a flex value of 1, the second box has a flex value of 2, and the third box has a flex value of 3. This means the third box will take up twice as much space as the first box, and three times as much space as the second box.

Stylesheet​

We can add the style object directly to our components using the style attribute. We can also use a stylesheet to store all of our styles in one place. Then, we can import that stylesheet into our React component and apply the styles using class names.

When a component is created, it can be assigned a style rule from the stylesheet. This means that the component will have the styles defined in that rule applied to it. Additionally, a component can have multiple style rules applied to it, allowing it to have a combination of different styles.

For example, you might have a stylesheet that says all the headings should be big and red, all the paragraphs should be small and black, and all the buttons should be green and rounded. When you apply the stylesheet to your React component, the component will have the styles defined in the stylesheet applied to it. This way, the website or app looks consistent and organized.

In the example below, we have a stylesheet that defines three styles: container, bigBlue, and red. The container style is applied to the View container, and the bigBlue and red styles are applied to the Text components. The bigBlue style is applied to the first Text component, and the red style is applied to the second Text component. The third Text component has both the bigBlue and red styles applied to it, and the fourth Text component has both the red and bigBlue styles applied to it. This means the third Text component will have the bigBlue style applied to it first, and then the red style will be applied to it. The fourth Text component will have the red style applied to it first, and then the bigBlue style will be applied to it.

This example uses a StyleSheet to make Text components with different styles. Now, it is your turn to apply StyleSheet to your text and shapes!
